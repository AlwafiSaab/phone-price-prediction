# 📱 Phone Price Prediction — Random Forest

> Predicting mobile phone price ranges using machine learning.
> Built by **Ahmed Al-Wafi** — BI & AI Automation Analyst

---

## 🎯 Project Overview

This project builds a machine learning model to predict the **price range of mobile phones** based on their technical specifications.

The model helps businesses and analysts understand which features drive phone pricing — useful for procurement, market analysis, and competitive benchmarking.

---

## 📊 Model Performance

| Metric | Result |
|--------|--------|
| Algorithm | Random Forest |
| Cross-Validation Score (Train) | **0.91 ± 0.01** |
| Output | Top 5 predicted price ranges by Phone ID |

> ✅ The model correctly predicts phone price range **91% of the time** across all validation folds.

## 🔄 Project Workflow

1. **Load & Explore** — Understand features and distributions
2. **Clean & Prepare** — Handle missing values, prepare features
3. **Train Model** — Random Forest with cross-validation
4. **Evaluate** — Achieved CV Score of 0.91 ± 0.01
5. **Predict** — Output top 5 price range predictions by phone ID
